# scrimba-tailwind-learning-path

A comprehensive collection of projects completed during the Scrimba [Learn Tailwind CSS course](https://scrimba.com/learn-tailwind-css-c010). This repository showcases hands-on learning through building real-world applications with HTML and Tailwind CSS.

## About This Repository
This repository contains all the projects and challenges I completed while working through Scrimba's Learn Tailwind CSS course. Each challenge/project demonstrates different skills and concepts essential for modern development.

## Projects
**All the designs have both mobile and desktop layouts
  
|Project  |Description  |
|:---------|-------------|
|Pomodoro App|A simple mock-up of a pomodoro timer|
|Testimonials Section|A simple mock-up of paw-testimonials from cat owners|
|Pricing Page|A pricing page that advertises three subscription tiers|
|Preferences Form|A simple tea ordering form with delivery & payment options|
|404 Error Page|An example of what would be displayed to the user in the case of a 404 error|
